# Git repository rules

1. Use semantic commit messages: https://gist.github.com/joshbuchea/6f47e86d2510bce28f8e7f42ae84c716
2. Semantic massage does not include any thing before `:`, it's place fro just type of change. 

It's forbidden:

```text
docs(sprint-14): add functional tests documentation and snippet status table
```

It's correct:

```text
docs: (sprint-14) add functional tests documentation and snippet status table
```
